# AL AMEEN ERP — دليل التشغيل والبيع

---

## 🖥️ تشغيل كتطبيق محلي (PWA)

### الطريقة الأسهل: Chrome/Edge
1. افتح `index.html` في Chrome أو Edge
2. ستظهر رسالة "تثبيت الآن" في أسفل الشاشة
3. اضغط "تثبيت الآن" — سيظهر كأيقونة على الـ Desktop
4. يعمل **بدون إنترنت** بعد أول تشغيل

### iOS (iPhone/iPad)
1. افتح في Safari
2. اضغط زر المشاركة → "إضافة إلى الشاشة الرئيسية"
3. سيظهر كتطبيق حقيقي

### تشغيل على شبكة محلية (LAN)
لتشغيله على أكثر من جهاز في نفس الشبكة:

```bash
# تثبيت serve (مرة واحدة)
npm install -g serve

# تشغيل السيرفر
cd AL_AMEEN_ERP
serve -s . -l 8080

# افتح من أي جهاز في الشبكة:
# http://192.168.1.x:8080
```

## 🏢 إنشاء شركة جديدة

1. سجل الدخول بحساب admin
2. في شاشة اختيار الشركة → "+ شركة جديدة"
3. أدخل بيانات الشركة وارفع اللوجو
4. كل شركة لها قاعدة بيانات منفصلة تماماً

---

## 📦 نقل البيانات بين أجهزة

**التصدير:**
- داخل الشركة → الإعدادات → "⬇ تصدير"
- سيتم حفظ ملف: `ALAMEEN_اسم_الشركة_YYYY-MM-DD.json`

**الاستيراد:**
- في الجهاز الجديد، افتح نفس الشركة → الإعدادات → "⬆ استيراد"

**نقل كامل (شجرة الحسابات + كل الشركات):**
- انسخ محتوى `localStorage` يدوياً من DevTools أو استخدم الـ export لكل شركة

## 🌐 نظام البيع كـ SaaS (سيرفر سحابي)

لتحويل البرنامج لمنتج تبيعه لعملاء متعددين عبر الإنترنت:

### الخطوات:

#### 1. رفع الملفات على سيرفر
```bash
# مثال: Nginx على Ubuntu
sudo apt install nginx
cp -r AL_AMEEN_ERP/* /var/www/html/alameen/

# أو استخدم Netlify / Vercel مجاناً
# فقط ارفع الملفات وستحصل على رابط
```

#### 2. استخدام Firebase كـ Backend
```javascript
// بدّل localStorage بـ Firestore
// كل شركة = document منفصل
// كل مستخدم = auth account
```

#### 3. هيكل قاعدة البيانات السحابية المقترح
```
/users/{userId}          → بيانات المستخدم + صلاحياته
/companies/{companyId}   → بيانات الشركة
/data/{companyId}/       → بيانات كل شركة منفصلة
  ├── accounts[]
  ├── journal[]
  ├── invoices[]
  └── ...
```

#### 4. ما تحتاجه للبيع
- دومين: مثل `alameen-erp.com`
- SSL: مجاني عبر Let's Encrypt
- Firebase: الخطة المجانية تكفي لأول 50 شركة
- أو استخدم Supabase كبديل مفتوح المصدر
